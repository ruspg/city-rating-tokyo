# City Rating: Tokyo Neighborhood Explorer

## Technical Specification v1.0

---

## 1. Product Overview

**Цель:** Публичный интерактивный сайт-карта Большого Токио с рейтингами районов вокруг станций. Пользователь видит карту с 200-300 станциями, кликает на станцию и получает: описание района, weighted score, цены на аренду, время до центров города.

**Целевая аудитория:** Иностранцы и экспаты, выбирающие район для жизни в Токио.

---

## 2. Ключевые фичи

### 2.1 Интерактивная карта
- Leaflet.js + CartoDB Positron tiles (бесплатно, без API-ключа)
- 200-300 маркеров станций Greater Tokyo (23 special wards + Yokohama, Kawasaki, Saitama ключевые)
- Маркеры цветом отражают общий рейтинг (gradient: красный=низкий -> зеленый=высокий)
- Кластеризация на малом зуме (Leaflet.markercluster)
- Popup при клике: краткая сводка + ссылка на полную карточку

### 2.2 Карточка станции/района
- Живое описание района (AI-generated, проверенное)
- Weighted score с разбивкой по категориям
- Средняя аренда (1K/1LDK/2LDK, 40-50 sqm)
- Время до ключевых центров (Shibuya, Shinjuku, Tokyo, Ikebukuro, Shinagawa)
- Фото/визуалы (опционально, v2)

### 2.3 Фильтры и сортировка
- Боковая панель с фильтрами: бюджет аренды, макс. время до центра, мин. рейтинг
- Настраиваемые веса категорий рейтинга (слайдеры)
- Сортировка таблицы по любой метрике

### 2.4 Weighted Score система

**Категории рейтинга (1-10 каждая):**

| Категория | Описание | Вес по умолчанию |
|-----------|----------|-----------------|
| `food` | Еда и рестораны, разнообразие кухни | 15% |
| `nightlife` | Бары, изакаи, ночная жизнь | 10% |
| `transport` | Доступность линий, время до центров | 20% |
| `rent` | Доступность аренды (инверсия: дешевле = выше) | 20% |
| `safety` | Безопасность района | 10% |
| `green` | Парки, зелень, прогулки | 10% |
| `gym_sports` | Спортзалы, фитнес-клубы рядом | 5% |
| `vibe` | Общая атмосфера, "душевность" района | 5% |
| `crowd` | Загруженность (инверсия: тише = выше) | 5% |

**Итоговый score** = сумма(категория * вес) / сумма(весов)

Пользователь может двигать слайдеры весов — карта перерисовывается в реальном времени.

## 4. Источники данных

### 4.1 Список станций и координаты

**Первичный:** [piuccio/open-data-jp-railway-stations](https://github.com/piuccio/open-data-jp-railway-stations) — предобработанный JSON, EN/JP имена, координаты, линии.

**Дополнительно:** [ekidata.jp](https://ekidata.jp) — CSV с кодами станций (нужны для маппинга на Suumo `ek` параметр).

**Фильтрация:** `pref_cd` = 13 (Tokyo), 14 (Kanagawa), 11 (Saitama), 12 (Chiba). Из ~600 записей отбираем 200-300 значимых станций (по пассажиропотоку или по количеству линий).

### 4.2 Данные аренды

**Основной подход: Suumo scraping**

URL-паттерн поиска:
```
https://suumo.jp/jj/chintai/ichiran/FR301FC001/?ar=030&ta=13&sc={ward_code}&ek={station_code}&mb=40&mt=55&md=02&md=04&md=07&pc=50
```

Параметры для нашего кейса:
- `mb=40&mt=55` — площадь 40-55 кв.м
- `md=02&md=04&md=07` — 1K + 1LDK + 2LDK
- `smk=r10` — в 10 минутах ходьбы от станции

**Инструменты:** `requests` + `BeautifulSoup` (Python). Suumo рендерит HTML server-side. Задержка 2-3 сек между запросами. Парсим `<div class="cassetteitem">` блоки.

**Альтернатива:** REINFOLIB API (MLIT) — бесплатный официальный API с данными сделок по станциям. Требует регистрации (~5 рабочих дней). Endpoint: `https://www.reinfolib.mlit.go.jp/ex-api/external/XIT001`

### 4.3 Время до центров

**AI-оценка** — Claude оценивает на основе знаний о линиях и пересадках. Для 200-300 станций это быстрее и дешевле API. Точность: +/- 5 минут, достаточно для рейтинга.

Целевые центры: Shibuya, Shinjuku, Tokyo, Ikebukuro, Shinagawa.

### 4.4 Описания районов

**AI research loop** — для каждой станции Claude генерирует описание по шаблону:
1. Атмосфера района
2. Достопримечательности и прогулки
3. Еда и кафе
4. Бары и вечерние места
5. Загруженность (1-10)

Плюс дополнительно:
6. Спортзалы / фитнес
7. Парки в пешей доступности
8. Общая оценка по каждой категории рейтинга (1-10)

## 5. Пайплайн наполнения данными

### Phase 1: Подготовка базы (1-2 дня)
1. Скачать и отфильтровать станции из piuccio/ekidata
2. Выбрать 200-300 станций (критерий: >=2 линий или на Yamanote/Chuo/Metro)
3. Сгенерировать `stations.json` с координатами

### Phase 2: AI Research Loop (итеративно, ~5-10 станций за сессию)
Для каждой станции:
1. Claude получает название станции
2. Генерирует описание + рейтинги по категориям
3. Оценивает время до 5 центров
4. Результат сохраняется в `data/ratings/{slug}.json`
5. Человек review + правки

**Оценка трудозатрат:** 200 станций / 10 за сессию = ~20 сессий

### Phase 3: Scrape аренды (1-2 дня)
1. Скрипт `scrape-suumo.py` обходит Suumo по каждой станции
2. Собирает 10-30 листингов на станцию
3. Считает медиану для 1K/1LDK и 2LDK
4. Сохраняет в `data/rent/rent-averages.json`

### Phase 4: Сборка и деплой (2-3 дня)
1. Next.js фронтенд с SSG (статическая генерация страниц станций)
2. Карта Leaflet
3. Фильтры и weighted score
4. Деплой на Vercel

## 8. Нефункциональные требования

- **Перформанс:** Карта с 300 маркерами загружается < 3 сек
- **SEO:** Каждая станция — отдельная SSG-страница с мета-тегами
- **i18n:** EN как основной, JP как secondary
- **Mobile:** Responsive, карта на весь экран на мобильном, фильтры в drawer
- **Данные:** JSON в репозитории, обновление через скрипты + PR
- **Хостинг:** Vercel free tier (достаточно для статики)

---

## 9. Риски и митигации

| Риск | Вероятность | Митигация |
|------|-------------|-----------|
| Suumo заблокирует scraping | Средняя | Fallback на REINFOLIB API или AI-оценку средних цен |
| AI-описания неточные | Средняя | Человеческий review каждой станции |
| 200+ станций — долго наполнять | Высокая | Начать с 50 станций Yamanote + Metro core, расширять итеративно |
| Suumo station codes не маппятся на ekidata | Низкая | Маппинг через название + линию, ручная коррекция |

## 10. MVP Scope (v1)

**Включено:**
- 50-70 станций (Yamanote line + ключевые Metro)
- Карта с цветными маркерами
- Weighted score с настраиваемыми весами
- Описания районов (AI + review)
- Средняя аренда (Suumo scrape)
- Время до 5 центров (AI-оценка)
- SSG-страницы для SEO
- EN язык

**Не включено в v1:**
- Японский язык (v2)
- Фото районов (v2)
- Пользовательские отзывы (v2)
- Сравнение районов side-by-side (v2)
- Полное покрытие 200-300 станций (постепенное расширение)
